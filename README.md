## Now Open for Community Contributions! 🎉

UnderSheet is fully open-source (MIT license) — **anyone can help make it better**!

The `main` branch is **protected**: no direct pushes, deletions, or force-pushes allowed. All changes (even small ones from me) must come through **pull requests (PRs)**. This keeps the core safe while letting the community add value gradually.

**Especially welcome:**
- New platform adapters (e.g., Lemmy, Bluesky, Mastodon, forums — just follow the adapter template)
- Bug fixes or better error handling
- Documentation improvements, examples, or usage tips
- Tests, performance tweaks, or proxy/VPN edge cases
- Anything that makes threading/memory easier for agents

### Quick Contribution Guide
1. **Fork** the repo (or clone if you have write access)
2. Create a branch: `git checkout -b feature/my-cool-platform` (or `fix/bug-description`)
3. Make your changes
4. Commit & push to your branch/fork
5. Open a **Pull Request** to `main`
6. Describe what you changed and why — I'll review and merge good ones quickly

See the full adapter guide in [Adding a Platform](#adding-a-platform) below.

Questions or ideas? Open an issue, DM me on X [@A2091_](https://x.com/A2091_), or just PR it!

Thanks to everyone who's checked it out — excited to see UnderSheet grow with community help! 🚀
